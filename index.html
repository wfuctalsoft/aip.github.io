AIP - новый формат файла, который поможет вам делиться приложениями и файлами без усобых усилий.

AIP-пакет может хранить огромные иерархии файлов любого размера, с его помощью вы можете создать или установить любые файлы в пару кликов.

Для использования AIP-пакетов была создана бесплатная программа AIPmini. Минималистичность и простота позволят любому познакомиться с AIP-пакетами.

Как использовать AIPmini?

Скачать AIPmini можно ниже, нажав кнопку "Скачать". После установки рекомендуется 1 раз открыть AIPmini.

Установка пакета:

Чтобы установить пакет, после установки AIPmini нужно всего 2 раза щелкнуть по файлу. Если пакет того требует, нужно прочитать и согласиться с лицензией.

Создание пакета:

Сначала нужно переместить все нужные вам файлы в одну папку, которую можно назвать как удобно (такая папка - это главная директория). Если вам не нужна настройка, то нажмите правой кнопкой мыши по главной директории, откроется контекстное меню, затем выберите пункт "Создать AIP пакет"

Чтобы провести настройку, создайте в главной директории файл с названием "Package.aini", затем откройте его с помощью любого текстового редактора.

Синтаксис команды:

Пункт = значение

Чтобы изменить описание пакета, используйте пункт "Description":

Description = Тут может быть ваше описание

Чтобы настроить производителя, используйте пункт "Manufacturer":

Manufacturer = Иван Иванов

Чтобы настроить путь к ярлыку, который будет создан на рабочем столе, используйте пункт "Link":

Link = Картинка\Картинка.png

Стоит заметить, что путь к файлу задается относительно главной директории. Если ярлык не нужен, значение должно равняться "None", например:

Link = None

Если файл находится по пути "C:\Users\Music\Главная Директория\Музыка.mp3" где "C:\Users\Music\Главная Директория" это путь к главной директории, то относительный путь равен "Музыка.mp3"

Чтобы добавить лицензионное соглашение, используйте пункт "EULA":

EULA = Ваше лицензионное соглашение

Сохраните ваш файл и создайте пакет через контекстное меню (указано выше)
